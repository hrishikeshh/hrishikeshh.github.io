Publications

  • Aug.2018
    Empirical Analysis of Bitcoin Market Volatility Using Supervised Learning Approach
    IEEE
    Based on financial econometrics of cryptocurrency's volatility using the traditional machine learning approaches. In this paper empirical testing of four conventional machine learning methods is performed to predict the bitcoin prices using last eight years of transactional data. Linear and polynomial regression is implemented using all the features individually. Polynomial regression, Support Vector regression and KNN regression are hyper tuned with grid search logic. Results depicted that KNN regression outperformed others models
